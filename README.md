⚔️ How to Run the Tools (Activation Guide)

🌟 “To unlock the power within these tools, follow the ritual below…”

🧪 Step 1: Check Your Power (Java Installed?)

Open terminal and type:

java --version

If Java is installed, your system is ready ⚡

If not, install JDK first.

🚀 Step 2: Summon the Tool

Navigate to the folder containing the .jar file:

cd folder-name

Then activate:

java -jar fileName.jar

Example:

java -jar ResumeBuilder.jar
🎯 And That’s It!

The tool will launch like a skill being unleashed 💥

“Every command typed is a step toward mastery.”
